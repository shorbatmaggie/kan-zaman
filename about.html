About the Tool

Kﾄ］ Zamﾄ］ is a CE to AH date converter for academics working on the Islamicate world. It aims to improve upon existing tools through the following:

  • Accessibility: a toggle for dyslexic-friendly fonts
  • Date Flexibility: the ability to choose between conversions for an entire year range or a specific day
  • Language: support for Arabic inputs; English, transliterated, and Arabic outputs
  • Updated UI: a modern and uncluttered display

Acknowledgments

  • This tool adapts the converter created by John Walker.
  • Transliterations follow IJMES guidelines.
  • The Dyslexia-friendly font is Abbie Gonzalez's OpenDyslexic. Learn more about her work here.
